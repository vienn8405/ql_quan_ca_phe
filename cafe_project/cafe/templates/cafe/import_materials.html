Nhập nguyên vật liệu từ Excel

Upload file Excel (.xlsx) để thêm mới hoặc cập nhật nguyên vật liệu hàng loạt.

← Quay lại kho Tải file mẫu Excel

Hướng dẫn

  • Tải file mẫu Excel bằng nút Tải file mẫu Excel ở trên.
  • Điền dữ liệu theo đúng cột: Tên nguyên liệu, Đơn vị, Số lượng, Mức tối thiểu.
  • Nếu chọn chi nhánh, số lượng và mức tối thiểu sẽ được cập nhật cho kho nguyên liệu của chi nhánh đó.
  • Nếu không chọn chi nhánh, dữ liệu sẽ được cập nhật vào kho dùng chung để giữ tương thích với dữ liệu cũ.
  • Chỉ hỗ trợ file .xlsx.
